# pdf-generator-kit の使用方法

このフォルダは、既存の原稿プロジェクトに「高品質なPDF変換機能」を後付けするためのセットコードです。
以下の **1〜4 のステップ** を順に進めることで、執筆した原稿をプロ仕様のPDFとして出力できるようになります。

---

## 1. 導入（セットアップ）

まずは、自分のプロジェクトにこのキットを組み込みます。

1. **キットを配置する**
   原稿（.md ファイル）が置かれている一番上のフォルダ（プロジェクトルート）に、この `pdf-generator-kit` フォルダをコピーして配置します。
   
   ```text
   自分のプロジェクトルートフォルダ/
     ├── (あなたの原稿など)
     └── pdf-generator-kit/  <-- フォルダごと置く
   ```

2. **セットアップを実行する**
   配置したフォルダの中にある `setup.sh` を実行します。
   （※ ターミナルで `bash pdf-generator-kit/setup.sh` と打つか、環境によってはダブルクリックで実行可能です）

3. **完了を待つ**
   必要な設定や資材がプロジェクトルートに自動展開されます。既存のファイルを壊すことはありません。

---

## 2. まず動作を確認する（チュートリアル）

セットアップが正しくできたか、付属のサンプルを使って確認してみましょう。

1. **変換コマンドを実行する**
   ターミナルで以下のコマンドを入力して Enter を押してください。
   ```bash
   npm run pdf articles/SAMPLE.md
   ```
2. **PDFを確認する**
   新しく作られた `dist/pdf/` フォルダの中に、`SAMPLE.pdf` が出来ていれば成功です！

   ```text
   自分のプロジェクトルートフォルダ/
     └── dist/
         └── pdf/
             └── SAMPLE.pdf  <-- ここに出力される
   ```

---

## 3. 原稿を書く

動作が確認できたら、いよいよ自分の教材を執筆します。

1. **執筆の見本を見る**
   `articles/SAMPLE.md` を開いてください。これが**実際の執筆マニュアル**を兼ねています。見出しの書き方や改ページ、コラムの作り方などがすべて網羅されています。
2. **画像を準備する**
   画像を使う場合は、原稿と同じ階層にある `images/` フォルダの中に保存し、相対パスで指定します（書き方は SAMPLE 参照）。

   ```text
   自分のプロジェクトルートフォルダ/
     ├── my-article.md (原稿)
     └── images/
         └── sample-image.png (画像データ)
   ```

---

## 4. PDFを完成させる

原稿が書けたら、いよいよPDFに変換します。

1. **コマンドを入力する**
   ターミナルで `npm run pdf` と打った後に、**「自分の原稿ファイルまでの道のり（パス）」** を続けて入力します。

   **【入力例】**
   *   `npm run pdf my-manuscript.md` （ファイルがすぐ隣にある場合）
   *   `npm run pdf articles/chapter1.md` （articlesフォルダの中にある場合）

2. **PDFを受け取る**
   変換されたファイルは、常に `dist/pdf/` フォルダに出力されます。

   ```text
   自分のプロジェクトルートフォルダ/
     └── dist/
         └── pdf/
             └── my-manuscript.pdf (完成！)
   ```

---

## こだわり調整（カスタマイズ）

プロジェクトに合わせてデザインを変えたい場合は、`styles/style.css` を編集します。
- `root` 部分の数値を書き換えるだけで、テーマカラーを一括変更できます。
- フォントの種類やサイズもこのファイルで自由に変更可能です。

---

## AIアシスタントへの指示

制作中にAIに相談する場合は、以下のように伝えてください。
> 「このプロジェクトの制作ルール（SKILL.md）を教えて」

キットが展開した `SKILL.md` をAIが読み込むことで、このキット特有のルールに基づいた的確なサポートを受けることができます。
